Fast Algorithms for Generating Integer Partitions

This repository hosts research papers on fast algorithms for generating integer partitions, with emphasis on efficient generation orders and state-of-the-art performance.

Background: ZS Algorithms

The paper “Fast algorithms for generating integer partitions – International Journal of Computing Mathematics, Vol. 70, pp. 319–332” describes two fast algorithms, ZS1 and ZS2, for generating integer partitions in standard representation.

Algorithm ZS1 generates partitions in anti-lexicographic order, while ZS2 generates partitions in lexicographic order. It has been observed by many professionals that ZS1 represents the state-of-the-art in fast generation of integer partitions.

Accelerated Algorithms: Z Series

The original paper “Algorithm Z1 Accelerated and Parallel Generation of Integer Partitions” introduces new algorithms Z1 (formerly AZ1) and Z2 (formerly AZ2) built upon the ZS1 and ZS2 framework. verion 2.1 introduces some enhencement to Z1, and introduces Algorithm Z3 to generate fairly distributed partitions subsets for parallel execution on single hardware. Time saving up to 89%.

  • Z1 – acceleration of algorithm ZS1
  • Z2 – acceleration of algorithm ZS2
  • Z3 – Generate faily distributed partitions subsets

These algorithms further improve performance while preserving the correctness and ordering properties of the original ZS methods. Algorithm Z1 claims time saving on ZS1 up to 35% on single execution, and up to 89% on Multi-Threaded executions on single hardware system.
